## Why This Role Matters

Nexus is building a civic-tech platform to make verified public data accessible in real time—helping users engage with accurate, trustworthy information when it matters most. We're laying the foundation for a suite of open-access dashboards spanning governance, housing, transit, health, and more—all built on a unified architecture designed for long-term public value.

## Your Mission

Own full-stack development including architecture, backend APIs, frontend interfaces, authentication, mapping, and deployment. Collaborate with designers and provide feedback to help shape the product vision.

## Stack

-   **frontend:** React, Next.js, TypeScript, Tailwind CSS
-   **backend:** Node.js, Express, GraphQL
-   **database:** DynamoDB, PostgreSQL
-   **hosting:** AWS Amplify, Netlify, S3 + CloudFront
-   **mapping:** Mapbox GL JS, Leaflet
-   **auth:** AWS Cognito, Firebase Auth
-   **versioning:** GitHub
